# Habit Tracker App

## Technologies Used

1.  NodeJS
2.  Express
3.  EJS
4.  MongoDB
5.  Javascript
6.  Mongoose

## Prerequisites

- Javascript
- NodeJS
- MongoDB
- Git

## Project Installation Guide

##### First Clone the Repository

`git clone `

##### Move to the project directory

`cd habit-tracker-app`

##### Next Install NPM dependencies

`npm install`

##### Next start your app

`npm start`

#### Your Server should now be running on http://localhost:9000/
